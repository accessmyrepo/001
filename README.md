# Datasets Used

This project utilizes the following publicly available image forgery detection datasets.
This repository includes processed versions of publicly available datasets (CASIA, CoMoFoD, CG-1050) used strictly for academic and research purposes.
All original datasets are the property of their respective authors and institutions.
The datasets have been reorganized and preprocessed for the purpose of our research.
This repository does not claim ownership of the original data and it's only for academic use.

## 1. CASIA v1.0 and CASIA v2.0
https://github.com/namtpham/casia2groundtruth
### Citation:
J. Dong, W. Wang, and T. Tan, “CASIA Image Tampering Detection Evaluation Database,” in Proceedings of the 2013 IEEE China Summit and International Conference on Signal and Information Processing (ChinaSIP), 2013, pp. 422–426.

## 2. CG-1050 Dataset
